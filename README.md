### 写这本书

为什么要写这本书呢？一方面是为我，我希望把我在学习数学中遇到的所有我认为好用的方法记录下来，或许在解题时可以用到。另一方面，就是为所有在数学方面有困惑的人施以援手，正如这本书的副标题所说，“Mathematics Lifesaver”，数学的救生员嘛。

### 这本书的标题

这本书的标题的确有些冗长了，完整翻译是
`高效的数学 - 初中数学的救生员：你需要掌握数学的所有工具`
主标题是来自我读过的一本书叫`Effictive Java`，我沿用了这本书的编排，将所有的数学技巧分条列出，这样编排很好。
